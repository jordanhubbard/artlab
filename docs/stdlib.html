stdlib/math

import { lerp, clamp, ... } from '../../src/stdlib/math.js'

Constants

NameValueDescription
DEG_TO_RADMath.PI / 180Multiply degrees by this to get radians.
RAD_TO_DEG180 / Math.PIMultiply radians by this to get degrees.

Three.js re-exports

Re-exports Vector2, Vector3, Vector4, Matrix3, Matrix4, Quaternion, and MathUtils from Three.js with short aliases Vec2, Vec3, Vec4, Mat3, Mat4, Quat.

deg(r) → num

Convert radians to degrees.

rad(d) → num

Convert degrees to radians.

clamp(v, min, max) → num

Clamp v to [min, max].

lerp(a, b, t) → num

Linear interpolation from a to b by t.

map(v, inMin, inMax, outMin, outMax) → num

Re-map v from one numeric range to another. Does not clamp output.

smoothstep(edge0, edge1, x) → num

GLSL-style smoothstep; returns 0 below edge0, 1 above edge1, smooth S-curve in between.

Easing functions → num

All accept a single t in [0, 1] and return a value in [0, 1].

FunctionDescription
easeIn(t)Quadratic ease-in (accelerate).
easeOut(t)Quadratic ease-out (decelerate).
easeInOut(t)Quadratic ease-in-out.
elasticOut(t)Elastic ease-out with one overshoot.
bounceOut(t)Bounce ease-out (multiple bounces).

noise2(x, y) → num

Perlin-style value noise for 2D input. Returns a value in [-1, 1].

noise3(x, y, z) → num

Perlin-style value noise for 3D input. Returns a value in [-1, 1].

Geometry factories

Each factory returns a THREE.BufferGeometry ready to pass to mesh().

FunctionParametersDefaultsDescription

box(w, h, d) w: num, h: num, d: num 1, 1, 1 Box geometry.

torus(R, r, ts, rs) R: num, r: num, ts: num, rs: num 1, 0.3, 64, 16 Torus; R = major radius, r = tube radius.

ring(iR, oR, segs) iR: num, oR: num, segs: num 0.5, 1, 64 Ring/disc; iR = inner radius, oR = outer radius.

mesh(geometry, options) → THREE.Mesh

ParameterTypeDefaultDescription
geometryTHREE.BufferGeometry—Geometry to wrap.
options.colornum | str0xffffffHex number or CSS color string.
options.roughnessnum0.7PBR roughness (0–1).
options.metalnessnum0.0PBR metalness (0–1).
options.wireframeboolfalseRender as wireframe.

Wrap a geometry in a MeshStandardMaterial and return the THREE.Mesh.

add(scene, geometry, options) → THREE.Mesh

Create a mesh and immediately add it to the scene. Same options as mesh().

import { sphere, mesh, add } from '../../src/stdlib/geometry.js'

export function setup(ctx) {
  const earth = ctx.add(mesh(sphere(1, 64), { color: 0x2266cc }))
}

stdlib/lights

import { ambient, point, directional, ... } from '../../src/stdlib/lights.js'

All factory functions return the corresponding THREE.Light object. Add it to the scene with ctx.add().

FunctionParameters (defaults)Description
ambient(color, intensity) 0x404040, 1 Flat ambient fill; no shadows.
point(color, intensity, distance, decay) 0xffffff, 1, 0, 2 Omnidirectional point light. distance=0 = infinite range.
directional(color, intensity) 0xffffff, 1 Parallel sun-style light. Shadow casting enabled, 2048×2048 map.
spot(color, intensity, distance, angle, penumbra) 0xffffff, 1, 0, Math.PI/4, 0.1 Cone spot light with shadow casting.
hemisphere(skyColor, groundColor, intensity) 0x88aaff, 0x332211, 1 Sky/ground gradient ambient light.

addLight(scene, light, position) → THREE.Light

Add a light to the scene and optionally set its position in one call.

import { directional, ambient } from '../../src/stdlib/lights.js'

export function setup(ctx) {
  const sun = ctx.add(directional(0xffffff, 2))
  sun.position.set(10, 20, 10)
  ctx.add(ambient(0x334455, 0.6))
}

stdlib/audio

import * as audio from '../../src/stdlib/audio.js'

start() → Promise<void>

Resume the audio context. Must be called from a user-gesture handler (click, keydown). Idempotent.

stop() → Promise<void>

Suspend the audio context. Call in teardown() if audio was started.

update() → { bass, mid, high, raw }

Analyse the latest audio frame. Call once per animation frame. Returns normalised band values (0–1) and the raw Float32Array FFT data.

band(name) → num

ParameterTypeDescription
name"bass" | "mid" | "high" | "treble"FFT band name ("high" and "treble" are aliases).

Return the current FFT band value (0–1) for a single named band.

play(src, options) → Promise<AudioBufferSourceNode>

ParameterTypeDefaultDescription
srcstr—URL or package-relative path.
options.volumenum1Linear gain (0–1+).
options.loopboolfalseLoop playback.
options.detunenum0Detune in cents.

Load and play a sound file. Requires start() to have been called first.

pad(options) → { setVolume(db), dispose() }

ParameterTypeDefaultDescription
options.notestr"C3"MIDI note name or frequency.
options.typestr"sine"Oscillator type.
options.reverbbooltrueRoute through engine reverb chain.

Create an ambient FM synth pad via Tone.js. Returns a handle with setVolume(db) and dispose().

Music helpers

FunctionReturnsDescription
scale(root, mode, octaves, startOctave)string[]Generate a note array for a named scale (e.g. 'minor', 'major').
chord(root, quality)string[]Return chord notes for a root and quality ('maj', 'min', 'dim', 'aug').
progression(root, str, mode, octave)string[][]Roman-numeral progression (e.g. 'I-IV-V-I') as arrays of note strings.

spatialize(audioNode, options) → { update(obj), disconnect() }

Link a Web Audio node to a Three.js object's world position via a PannerNode. Call update(obj) each frame to track the object's position. Call disconnect() to stop tracking.

Video input, file playback, and GPU shader effects for video textures.

FunctionReturnsDescription
webcam(opts?) { texture, video, start(), stop(), isReady() } Live webcam stream as a VideoTexture. Options: width, height, facingMode.
screen(opts?) { texture, video, start(), stop(), isReady() } Screen / tab capture as a VideoTexture. Options: width, height.

import { webcam, videoPlane } from '../../src/stdlib/video.js'

let cam

export async function setup(ctx) {
  cam = webcam()
  cam.start()
  const plane = ctx.add(videoPlane(cam))
  plane.position.set(0, 0, 0)
}

export function teardown() { cam.stop() }

stdlib/physics/particles

import { createParticleWorld, emitter, forceField } from '../../src/stdlib/physics/particles.js'

CPU particle simulation with optional force fields. For rigid-body and orbital physics see src/physics/ (Rapier3D wrappers).

createParticleWorld() → ParticleWorld

Create a new particle simulation world. Call world.step(dt) once per frame.

emitter(world, scene, options) → { id, dispose() }

OptionDefaultDescription
rate10Particles spawned per second.
lifetime2Particle lifetime in seconds.
speed1Initial speed in scene units/s.
spread30Cone spread in degrees.
color0xffffffParticle color (hex).
size0.1Particle point size.

Attach a particle emitter to the world. Manages its own THREE.Points mesh in the scene.

forceField(world, emitterId, center, radius, force) → void

Apply an attraction or repulsion force to particles within radius of center. Positive force = attraction.
