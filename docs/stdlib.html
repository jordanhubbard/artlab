ParameterTypeDescription

Classic Perlin-style value noise for 2D input. Returns a value in [-1, 1]. Uses a fixed permutation table (reproducible across runs).

artlab/geometry

DSL: use "artlab/geometry"

Geometry factories

Each factory returns a THREE.BufferGeometry ready to pass to mesh().

FunctionParametersDefaultsDescription
sphere(radius, detail) radius: num, detail: num 1, 32 Sphere geometry with detail×detail segments.
box(w, h, d) w: num, h: num, d: num 1, 1, 1 Box geometry.
cylinder(rt, rb, h, segs) rt: num, rb: num, h: num, segs: num 1, 1, 2, 32 Cylinder; rt = top radius, rb = bottom radius.
torus(R, r, ts, rs) R: num, r: num, ts: num, rs: num 1, 0.3, 64, 16 Torus; R = major radius, r = tube radius.
plane(w, h, ws, hs) w: num, h: num, ws: num, hs: num 1, 1, 1, 1 Plane geometry with optional segment counts.
ring(iR, oR, segs) iR: num, oR: num, segs: num 0.5, 1, 64 Ring/disc geometry; iR = inner radius, oR = outer radius.
cone(r, h, segs) r: num, h: num, segs: num 1, 2, 32 Cone geometry.

mesh(geometry, options) → THREE.Mesh

ParameterTypeDefaultDescription
geometryTHREE.BufferGeometry—Geometry to wrap.
options.colornum | str0xffffffHex number or CSS color string.
options.roughnessnum0.7PBR roughness (0–1).
options.metalnessnum0.0PBR metalness (0–1).
options.wireframeboolfalseRender as wireframe.

Wrap a geometry in a MeshStandardMaterial and return the resulting THREE.Mesh.

add(scene, geometry, options) → THREE.Mesh

ParameterTypeDescription
sceneTHREE.SceneScene to add the mesh to.
geometryTHREE.BufferGeometryGeometry to wrap.
optionsobjectSame options as mesh().

Create a mesh from geometry and immediately add it to the scene. Returns the mesh for further manipulation.

use "artlab/geometry" as geo

fn setup() {
  let earth = geo.add(scene, geo.sphere(1, 64), { color: 0x2266cc })
}

artlab/lights

DSL: use "artlab/lights"

All factory functions return the corresponding THREE.Light object.

FunctionParameters (defaults)Description
ambient(color, intensity) 0x404040, 1 Flat ambient fill light; no shadows.
point(color, intensity, distance, decay) 0xffffff, 1, 0, 2 Omnidirectional point light. distance=0 means infinite range.
directional(color, intensity) 0xffffff, 1 Sun-style parallel light. Shadow casting enabled, 2048×2048 shadow map.
spot(color, intensity, distance, angle, penumbra) 0xffffff, 1, 0, Math.PI/4, 0.1 Cone spot light. Shadow casting enabled.
hemisphere(skyColor, groundColor, intensity) 0x88aaff, 0x332211, 1 Sky/ground gradient ambient light.

addLight(scene, light, position) → THREE.Light

ParameterTypeDescription
sceneTHREE.SceneScene to add the light to.
lightTHREE.LightLight object from a factory.
position[x, y, z] (optional)Position array. Omit to keep the default position.

Add a light to a scene and optionally set its position in one call.

use "artlab/lights" as L

fn setup() {
  L.addLight(scene, L.directional(0xffffff, 2), [10, 20, 10])
  L.addLight(scene, L.ambient(0x334455, 0.6))
}

artlab/cameras

All camera factories return an object with at least { camera, update(dt), setTarget(obj), onResize() }.

OrbitCamera(renderer, opts) → { camera, controls, update, setTarget, onResize }

OptionTypeDefaultDescription
fovnum55Field of view in degrees.
nearnum0.1Near clip plane.
farnum200000Far clip plane.
minDistancenum2Minimum orbit radius.
maxDistancenum8000Maximum orbit radius.
zoomSpeednum1.2Scroll zoom speed.
rotateSpeednum0.5Mouse drag rotate speed.

Mouse-driven orbit camera using Three.js OrbitControls. Handles window resize automatically.

FlyCamera(renderer, opts) → { camera, update, setTarget, dispose, onResize }

OptionTypeDefaultDescription
speednum50Movement speed in units/second.
sensitivitynum0.002Mouse look sensitivity (radians/pixel).
fovnum55Field of view in degrees.

WASD + Q/E keyboard movement with hold-mouse-button look. Call dispose() to remove event listeners.

PathCamera(curve, opts) → { camera, update, setTarget, seek, t, autoPlay, onResize }

OptionTypeDefaultDescription
speednum0.1t-units per second (full range is 0→1).
autoPlaybooltrueAdvance automatically each frame.
startTnum0Initial position along the curve (0–1).
lookAheadnum0.01How far ahead on the curve to look.

Camera that follows a THREE.CatmullRomCurve3. Use seek(t) to jump to a position.

CinematicCamera(opts) → { camera, addKeyframe, seek, play, pause, update, setTarget, onResize, currentT, duration, playing }

OptionTypeDefaultDescription

Keyframe-driven camera. Add keyframes with addKeyframe(t, pos, target); positions are linearly interpolated between keyframes. Call play() to start and update(dt) each frame to advance playback.

artlab/animation

DSL: use "artlab/animation"

animations (AnimationSystem singleton)

Module-level singleton. All convenience functions register with this instance. Call animations.update(dt) once per frame from your render loop.

tween(target, to, duration, options) → Tween[]

ParameterTypeDefaultDescription
targetobject—Object owning the properties to animate.
toobject—Map of { propPath: targetValue }. Dot-notation is supported (e.g. "scale.x").
durationnum—Duration in seconds.
options.easingstr"linear"Easing name (e.g. "power2out").
options.delaynum0Delay before starting, in seconds.
options.onCompletefnnullCalled when all tweens in this call finish.

Create and start tweens on the global animation system. Returns an array of Tween objects; call .finish() on each to cancel early.

audioReact(target, prop, band, options) → { disconnect() }

ParameterTypeDefaultDescription
targetobject—Object owning the property.
propstr—Property path (dot-notation OK, e.g. "scale.y").
band"bass" | "mid" | "high"—FFT band to track.
options.minnum0Mapped minimum value.
options.maxnum1Mapped maximum value.

Create an audio-reactive binding that drives a property on every frame. Call .disconnect() to stop tracking.

timeline(keyframes) → { seek, play, pause, onComplete }

ParameterTypeDescription
keyframesArray<{ t: num, props: Map | object }>Sorted or unsorted keyframe array. Each keyframe maps target objects to { propPath: value }.

Build a keyframe timeline. Returns { seek(t), play(speed?), pause(), onComplete(cb) }. The timeline drives itself through the global animations tween system — no separate loop is needed.

use "artlab/animation" as anim

fn setup() {
  # tween mesh up over 2 seconds
  anim.tween(mesh.position, { y: 5 }, 2, { easing: "power2out" })

  # audio-reactive scale
  anim.audioReact(mesh.scale, "x", "bass", { min: 1, max: 3 })
}

fn update(dt: num) {
  anim.animations.update(dt)
}

artlab/audio

DSL: use "artlab/audio"

start() → Promise<void>

Resume the audio context. Must be called from a user-gesture handler (click, keydown). Idempotent.

update() → { bass, mid, high, raw }

Analyse the latest audio frame. Call once per animation frame. Returns normalised band values (0–1) and the raw Float32Array FFT data.

band(name) → num

ParameterTypeDescription
name"bass" | "mid" | "high" | "treble"FFT band name ("high" and "treble" are aliases).

Return the current FFT band value (0–1) for a single named band. Convenient for audio-reactive expressions.

play(src, options) → Promise<AudioBufferSourceNode>

ParameterTypeDefaultDescription
srcstr—URL or package-relative path.
options.volumenum1Linear gain (0–1+).
options.loopboolfalseLoop playback.
options.detunenum0Detune in cents.

Load and play a sound file. Requires start() to have been called first.

pad(options) → { setVolume(db), dispose() }

ParameterTypeDefaultDescription
options.notestr"C3"MIDI note name or frequency.
options.typestr"sine"Oscillator type.
options.reverbbooltrueRoute through engine reverb chain.

Create an ambient FM synth pad via Tone.js. Returns a handle with setVolume(db) (ramps over 1.5 s) and dispose().

spatialize(audioNode, options) → { update(obj), disconnect() }

ParameterTypeDefaultDescription
audioNodeAudioNode—Source node to spatialise.
options.refDistancenum1Distance at which attenuation begins.
options.maxDistancenum10000Distance at which gain is clamped.
options.rolloffFactornum1Rate of attenuation.
options.panningModelstr"HRTF""HRTF" or "equalpower".
options.distanceModelstr"inverse"Web Audio distance model.

Link a Web Audio node to a Three.js object's world position via a PannerNode. Call update(obj) each frame to track the object's position.

DSL: use "artlab/physics/orbital"

Constants

NameValueDescription

orbit(params) → { position(elapsed), params }

ParameterTypeDefaultDescription
params.semiMajorAxisnum—Semi-major axis in AU.
params.eccentricitynum0Orbital eccentricity (0 = circle).
params.inclinationnum0Inclination in degrees.
params.periodnum1Orbital period in Earth years.
params.phasenum0Initial mean anomaly offset in radians.

Create a Kepler orbit descriptor. Call .position(elapsedSeconds) to get the { x, y, z } position in scene units at any point in time.

attachOrbit(obj, orbitParams) → function(elapsed)

ParameterTypeDescription
objTHREE.Object3DObject to animate (writes to .position.x/y/z).
orbitParamsobjectSame parameters accepted by orbit().

Animate an object on a Keplerian orbit. Returns an updater function; call it each frame with elapsed seconds.

solarOrbit(planetName) → { position(elapsed), params }

ParameterTypeDescription
planetNamestrLower-case planet name: "mercury", "venus", "earth", "mars", "jupiter", "saturn", "uranus", "neptune".

Return an orbit descriptor pre-configured with real solar system data for the named planet.

use "artlab/physics/orbital" as orb

let earthUpdate = orb.attachOrbit(earthMesh, orb.solarOrbit("earth").params)

fn update(elapsed: num, dt: num) {
  earthUpdate(elapsed)
}

artlab/physics/bindings

DSL: use "artlab/physics/bindings"

createPhysics(options) → { composer, orbital, rigid, particles, step(elapsed, dt), dispose() }

OptionTypeDefaultDescription
gravitybooltruePre-apply Earth gravity (-9.81 Y) to the rigid world.

Create a unified physics context. All three sub-worlds (orbital, rigid, particles) are advanced with a single step(elapsed, dt) call.

attachOrbit(physicsCtx, obj, desc) → { id, detach() }

Attach a scene object to a Keplerian orbital body. After each step() the object's position is synced. Call detach() to remove.

attachRigid(physicsCtx, obj, desc) → { id, detach(), applyForce(vec3), applyImpulse(vec3) }

Desc fieldTypeDescription

shape{ type: "box" | "sphere" | ..., ...dims }Collision shape descriptor.
massnumMass in kg (dynamic bodies).
restitutionnumBounciness (0–1).

Attach a scene object to a Rapier rigid body. Position and quaternion are synced after each step.

createEmitter(physicsCtx, desc) → { id, getParticles(), detach() }

Desc fieldTypeDescription
ratenumParticles per second.
lifetimenumParticle lifetime in seconds.
speednumInitial speed in scene units/s.
spreadnumCone spread in degrees.
gravitynumDownward gravity factor.

Create a particle emitter. Retrieve live particles via getParticles() for rendering.

setGravity(physicsCtx, vec3) → void

ParameterTypeDescription

vec3{ x, y, z }Gravity in m/s². Earth = { x:0, y:-9.81, z:0 }.

Set the gravity vector applied to all dynamic rigid bodies.
