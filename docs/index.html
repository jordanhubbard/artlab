artlab
Browser IDE and runtime for generative 3D art & audio-reactive scenes.

What is Artlab?

Artlab is a browser-native IDE and runtime for building real-time 3D scenes, generative art, and audio-reactive visualizations with Three.js. Packages are plain ES modules — no compilation step, no custom language. Write setup(ctx), update(ctx, dt), and teardown(ctx); the IDE hot-reloads on every save.

The ctx object injected by the runtime gives you Three, scene, camera, renderer, controls, add(), setBloom(), and all stdlib helpers — so simple packages need no imports at all.

Quick Start

Clone the repo, install dependencies, and open the IDE:

# Node 18+, npm 8+
git clone <repo-url> artlab
cd artlab
npm install
npm run dev
# open http://localhost:5173

The IDE loads with the solar-system reference package in the preview pane. Open any example from the sidebar, edit a file, save — the preview updates instantly. Use the toolbar's Export button to download a package as a .zip.

The simplest possible package:

export function setup(ctx) {
  ctx.add(new ctx.Three.Mesh(
    new ctx.Three.BoxGeometry(1, 1, 1),
    new ctx.Three.MeshStandardMaterial({ color: 0x4488ff })
  ))
}

export function update(ctx, dt) {
  ctx.scene.children[0].rotation.y += dt
}
